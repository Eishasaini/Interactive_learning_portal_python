1. Operators

Operators are special symbols in Python that carry out arithmetic or logical computation. Python Operators in general are used to perform operations on values and variables.
These are standard symbols used for the purpose of logical and arithmetic operations.

In this tutorial, you'll learn everything about different types of operators in Python, their syntax and how to use them with examples.

Various Types of Operators:

Python has many types of Operators that are stated as below:

  • Arithmetic Operator
  • Logical Operator
  • Comparison Operator
  • Bitwise Operator
  • Assignment Operator
  • Identity Operator
  • Membership Operator
Arithmetic Operator
Python programming language supports different kinds of arithmetic operators for both integer and floating point like addition, subtraction, multiplication, division and so on.

Logical Operator
Logical operators are used for performing AND, OR and NOT operations. It either returns True or False according to the condition.

Comparison Operator

Comparison operators are used to compare values. It returns either True or False according to the condition.

Bitwise Operator

Bitwise operators act on operands as if they were strings of binary digits. They operate bit by bit, hence the name.

For example, 2 is 10 in binary and 7 is 111. In the table below: Let x = 10 (0000 1010 in binary) and y = 4 (0000 0100 in binary)

Assignment Operator

Assignment operators are used in Python to assign values to variables.

a = 5 is a simple assignment operator that assigns the value 5 on the right to the variable a on the left. There are various compound operators in Python like a += 5 that adds to the variable and later assigns the same. It is equivalent to a = a + 5.

Identity Operator

It comes under special operators.'is' and 'is not' are the identity operators in Python. They are used to check if two values (or variables) are located on the same part of the memory.
Two variables that are equal does not imply that they are identical.

Membership Operator

It comes under special operators.'in' and 'not in' are the membership operators in Python. They are used to test whether a value or variable is found in a sequence (string, list, tuple, set and dictionary).
In a dictionary we can only test for presence of key, not the value.
